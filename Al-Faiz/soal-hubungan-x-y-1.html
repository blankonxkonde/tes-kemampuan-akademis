Soal Hubungan X dan Y - Pembahasan

Konsep Hubungan X dan Y

Soal-soal hubungan X dan Y menguji kemampuan untuk menghitung nilai dari dua kuantitas (X dan Y) dan menentukan hubungan matematis antara keduanya. Hubungan yang dimaksud bisa berupa kesamaan (=), ketidaksamaan (>, <), atau hubungan khusus lainnya. Penting untuk menghitung dengan teliti dan memeriksa setiap opsi jawaban.

Soal Awal (Halaman Tidak Bernomor)

Soal 1
Diketahui
X Y
\(16 : \frac{1}{2} \times 1\frac{2}{5}\) \(12\frac{1}{2} + 48 \times 1\frac{1}{2} - 3\frac{1}{2}\)
Manakah hubungan yang benar antara kuantitas X dan Y berikut berdasarkan informasi yang diberikan? (TIU CPNS 2023)
  • A. \(X > Y\)
  • B. \(3X < Y\)
  • C. \(2X < 3Y\)
  • D. \(3X = 2Y\)
  • E. Hubungan X dan Y tidak dapat ditentukan
Pembahasan:
Menghitung nilai X:
\(X = 16 : \frac{1}{2} \times 1\frac{2}{5}\)
Perhatikan urutan operasi: pembagian dan perkalian memiliki prioritas yang sama, dikerjakan dari kiri ke kanan.
\(16 : \frac{1}{2} = 16 \times 2 = 32\)
\(1\frac{2}{5} = \frac{7}{5}\)
\(X = 32 \times \frac{7}{5} = \frac{224}{5} = 44,8\)
Menghitung nilai Y:
\(Y = 12\frac{1}{2} + 48 \times 1\frac{1}{2} - 3\frac{1}{2}\)
Perhatikan urutan operasi: perkalian dikerjakan terlebih dahulu.
\(12\frac{1}{2} = \frac{25}{2} = 12,5\)
\(1\frac{1}{2} = \frac{3}{2} = 1,5\)
\(3\frac{1}{2} = \frac{7}{2} = 3,5\)
\(48 \times 1\frac{1}{2} = 48 \times \frac{3}{2} = 72\)
\(Y = 12,5 + 72 - 3,5 = 81\)
Membandingkan X dan Y:
\(X = 44,8\)
\(Y = 81\)
Jadi \(X < Y\) atau \(Y > X\)
Memeriksa opsi jawaban:
A. \(X > Y\) → Salah (karena \(44,8 < 81\))
B. \(3X < Y\) → \(3(44,8) = 134,4\), \(134,4 < 81\)? Salah
C. \(2X < 3Y\) → \(2(44,8) = 89,6\), \(3(81) = 243\), \(89,6 < 243\)? Benar ✓
D. \(3X = 2Y\) → \(3(44,8) = 134,4\), \(2(81) = 162\), \(134,4 \neq 162\) → Salah
E. Hubungan tidak dapat ditentukan → Salah (karena sudah dapat ditentukan)
Jawaban: C. \(2X < 3Y\)
Soal 2
Diketahui
X Y
\(-\frac{3}{2} + \frac{11}{2} + 8 \times 4\) \(-\frac{9}{2} + \frac{3}{2} + 3 \times 8\)
Manakah hubungan yang benar antara kuantitas X dan Y berikut berdasarkan informasi yang diberikan? (TIU CPNS 2023)
  • A. \(2X = Y\)
  • B. \(3X = 4Y\)
  • C. Hubungan X dan Y tidak dapat ditentukan
  • D. \(3X < 2Y\)
  • E. \(2X > Y\)
Pembahasan:
Menghitung nilai X:
\(X = -\frac{3}{2} + \frac{11}{2} + 8 \times 4\)
Perhatikan urutan operasi: perkalian dikerjakan terlebih dahulu.
\(8 \times 4 = 32\)
\(-\frac{3}{2} + \frac{11}{2} = \frac{-3 + 11}{2} = \frac{8}{2} = 4\)
\(X = 4 + 32 = 36\)
Menghitung nilai Y:
\(Y = -\frac{9}{2} + \frac{3}{2} + 3 \times 8\)
Perhatikan urutan operasi: perkalian dikerjakan terlebih dahulu.
\(3 \times 8 = 24\)
\(-\frac{9}{2} + \frac{3}{2} = \frac{-9 + 3}{2} = \frac{-6}{2} = -3\)
\(Y = -3 + 24 = 21\)
Membandingkan X dan Y:
\(X = 36\)
\(Y = 21\)
Jadi \(X > Y\)
Memeriksa opsi jawaban:
A. \(2X = Y\) → \(2(36) = 72\), \(72 = 21\)? Salah
B. \(3X = 4Y\) → \(3(36) = 108\), \(4(21) = 84\), \(108 = 84\)? Salah
C. Hubungan tidak dapat ditentukan → Salah
D. \(3X < 2Y\) → \(3(36) = 108\), \(2(21) = 42\), \(108 < 42\)? Salah
E. \(2X > Y\) → \(2(36) = 72\), \(72 > 21\)? Benar ✓
Jawaban: E. \(2X > Y\)
Soal 3
Diketahui
X Y
\(-\frac{6}{4} \div (-\frac{6}{12}) - 4 \times 3\) \(-\frac{6}{5} + \frac{4}{5} - \frac{8}{5} \div \frac{2}{2}\)
Manakah hubungan yang benar antara kuantitas x dan y berikut berdasarkan informasi yang diberikan? (TIU CPNS 2023)
  • A. \(2x = y\)
  • B. \(\frac{1}{3}x > y\)
  • C. Hubungan x dan y tidak dapat ditentukan
  • D. \(3x > y\)
  • E. \(\frac{1}{3}x < y\)
Pembahasan:
Menghitung nilai X:
\(X = -\frac{6}{4} \div (-\frac{6}{12}) - 4 \times 3\)
Perhatikan urutan operasi: pembagian dan perkalian dikerjakan terlebih dahulu dari kiri ke kanan.
\(-\frac{6}{4} = -\frac{3}{2}\)
\(-\frac{6}{4} \div (-\frac{6}{12}) = -\frac{3}{2} \div (-\frac{1}{2}) = -\frac{3}{2} \times (-2) = 3\)
\(4 \times 3 = 12\)
\(X = 3 - 12 = -9\)
Menghitung nilai Y:
\(Y = -\frac{6}{5} + \frac{4}{5} - \frac{8}{5} \div \frac{2}{2}\)
Perhatikan: \(\frac{2}{2} = 1\)
Perhatikan urutan operasi: pembagian dikerjakan terlebih dahulu.
\(\frac{8}{5} \div 1 = \frac{8}{5}\)
\(-\frac{6}{5} + \frac{4}{5} = \frac{-6 + 4}{5} = -\frac{2}{5}\)
\(Y = -\frac{2}{5} - \frac{8}{5} = -\frac{10}{5} = -2\)
Membandingkan X dan Y:
\(X = -9\)
\(Y = -2\)
Jadi \(X < Y\) (karena \(-9 < -2\))
Memeriksa opsi jawaban:
A. \(2x = y\) → \(2(-9) = -18\), \(-18 = -2\)? Salah
B. \(\frac{1}{3}x > y\) → \(\frac{1}{3}(-9) = -3\), \(-3 > -2\)? Salah
C. Hubungan tidak dapat ditentukan → Salah
D. \(3x > y\) → \(3(-9) = -27\), \(-27 > -2\)? Salah
E. \(\frac{1}{3}x < y\) → \(\frac{1}{3}(-9) = -3\), \(-3 < -2\)? Benar ✓
Jawaban: E. \(\frac{1}{3}x < y\)
Soal 4
Diketahui
x y
\(-\frac{5}{4} + \frac{3}{4} + 2,5 \times \frac{1}{10}\) \(-\frac{1}{5} - \frac{3}{5} + 11 \times \frac{1}{20}\)
Manakah hubungan yang benar antara kuantitas x dan y berikut berdasarkan informasi yang diberikan?
  • A. \(2x = y\)
  • B. \(3x = 4y\)
  • C. Hubungan x dan y tidak dapat ditentukan
  • D. \(3x < 2y\)
  • E. \(2x > y\)
Pembahasan:
Menghitung nilai x:
\(x = -\frac{5}{4} + \frac{3}{4} + 2,5 \times \frac{1}{10}\)
Perhatikan urutan operasi: perkalian dikerjakan terlebih dahulu.
\(2,5 = \frac{5}{2}\)
\(2,5 \times \frac{1}{10} = \frac{5}{2} \times \frac{1}{10} = \frac{5}{20} = \frac{1}{4}\)
\(-\frac{5}{4} + \frac{3}{4} = \frac{-5 + 3}{4} = -\frac{2}{4} = -\frac{1}{2}\)
\(x = -\frac{1}{2} + \frac{1}{4} = -\frac{2}{4} + \frac{1}{4} = -\frac{1}{4}\)
Menghitung nilai y:
\(y = -\frac{1}{5} - \frac{3}{5} + 11 \times \frac{1}{20}\)
Perhatikan urutan operasi: perkalian dikerjakan terlebih dahulu.
\(11 \times \frac{1}{20} = \frac{11}{20}\)
\(-\frac{1}{5} - \frac{3}{5} = \frac{-1 - 3}{5} = -\frac{4}{5} = -\frac{16}{20}\)
\(y = -\frac{16}{20} + \frac{11}{20} = -\frac{5}{20} = -\frac{1}{4}\)
Membandingkan x dan y:
\(x = -\frac{1}{4}\)
\(y = -\frac{1}{4}\)
Jadi \(x = y\)
Memeriksa opsi jawaban:
A. \(2x = y\) → \(2(-\frac{1}{4}) = -\frac{1}{2}\), \(-\frac{1}{2} = -\frac{1}{4}\)? Salah
B. \(3x = 4y\) → \(3(-\frac{1}{4}) = -\frac{3}{4}\), \(4(-\frac{1}{4}) = -1\), \(-\frac{3}{4} = -1\)? Salah
C. Hubungan tidak dapat ditentukan → Salah
D. \(3x < 2y\) → \(3(-\frac{1}{4}) = -\frac{3}{4}\), \(2(-\frac{1}{4}) = -\frac{1}{2}\), \(-\frac{3}{4} < -\frac{1}{2}\)? Benar ✓
E. \(2x > y\) → \(2(-\frac{1}{4}) = -\frac{1}{2}\), \(-\frac{1}{2} > -\frac{1}{4}\)? Salah
Catatan:
Meskipun \(x = y\), opsi D benar karena \(3x = 3(-\frac{1}{4}) = -\frac{3}{4}\) dan \(2y = 2(-\frac{1}{4}) = -\frac{1}{2}\), dan \(-\frac{3}{4} < -\frac{1}{2}\) adalah benar.
Jawaban: D. \(3x < 2y\)
Soal 5
Diketahui
x y
\(-\frac{5}{4} \div (-\frac{5}{12}) - 5 \times 3\) \(-\frac{8}{5} + \frac{4}{7} - \frac{17}{5} \div \frac{7}{2}\)
Manakah hubungan yang benar antara kuantitas x dan y berikut berdasarkan informasi yang diberikan?
  • A. \(2x = y\)
  • B. \(3x = 2y\)
  • C. Hubungan x dan y tidak dapat ditentukan
  • D. \(x > y\)
  • E. \(\frac{1}{7}x < y\)
Pembahasan:
Menghitung nilai x:
\(x = -\frac{5}{4} \div (-\frac{5}{12}) - 5 \times 3\)
Perhatikan urutan operasi: pembagian dan perkalian dikerjakan terlebih dahulu dari kiri ke kanan.
\(-\frac{5}{4} \div (-\frac{5}{12}) = -\frac{5}{4} \times (-\frac{12}{5}) = \frac{60}{20} = 3\)
\(5 \times 3 = 15\)
\(x = 3 - 15 = -12\)
Menghitung nilai y:
\(y = -\frac{8}{5} + \frac{4}{7} - \frac{17}{5} \div \frac{7}{2}\)
Perhatikan urutan operasi: pembagian dikerjakan terlebih dahulu.
\(\frac{17}{5} \div \frac{7}{2} = \frac{17}{5} \times \frac{2}{7} = \frac{34}{35}\)
Untuk menjumlahkan \(-\frac{8}{5} + \frac{4}{7}\), samakan penyebutnya:
KPK dari 5 dan 7 adalah 35
\(-\frac{8}{5} = -\frac{56}{35}\)
\(\frac{4}{7} = \frac{20}{35}\)
\(-\frac{8}{5} + \frac{4}{7} = -\frac{56}{35} + \frac{20}{35} = -\frac{36}{35}\)
\(y = -\frac{36}{35} - \frac{34}{35} = -\frac{70}{35} = -2\)
Membandingkan x dan y:
\(x = -12\)
\(y = -2\)
Jadi \(x < y\) (karena \(-12 < -2\))
Memeriksa opsi jawaban:
A. \(2x = y\) → \(2(-12) = -24\), \(-24 = -2\)? Salah
B. \(3x = 2y\) → \(3(-12) = -36\), \(2(-2) = -4\), \(-36 = -4\)? Salah
C. Hubungan tidak dapat ditentukan → Salah
D. \(x > y\) → \(-12 > -2\)? Salah
E. \(\frac{1}{7}x < y\) → \(\frac{1}{7}(-12) = -\frac{12}{7} \approx -1,714\), \(-1,714 < -2\)? Salah

Periksa kembali opsi E:
\(\frac{1}{7}x = \frac{1}{7}(-12) = -\frac{12}{7} = -\frac{60}{35}\)
\(y = -2 = -\frac{70}{35}\)
\(-\frac{60}{35} < -\frac{70}{35}\)? Benar (karena \(-60 > -70\), maka \(-\frac{60}{35} > -\frac{70}{35}\))
Jadi \(\frac{1}{7}x > y\), bukan \(\frac{1}{7}x < y\)

Mari periksa apakah ada opsi yang benar:
Karena \(x = -12\) dan \(y = -2\), maka \(x < y\) atau \(y > x\).
Tidak ada opsi yang menyatakan \(x < y\) secara langsung.
Periksa: \(\frac{1}{7}x = -\frac{12}{7} \approx -1,714\) dan \(y = -2\)
Karena \(-1,714 > -2\), maka \(\frac{1}{7}x > y\), bukan \(\frac{1}{7}x < y\)

Kemungkinan ada kesalahan dalam perhitungan atau opsi E seharusnya \(\frac{1}{7}x > y\)
Namun berdasarkan opsi yang ada, tidak ada yang benar. Mari periksa ulang perhitungan y.
Verifikasi ulang perhitungan y:
\(y = -\frac{8}{5} + \frac{4}{7} - \frac{17}{5} \div \frac{7}{2}\)
\(= -\frac{8}{5} + \frac{4}{7} - \frac{17}{5} \times \frac{2}{7}\)
\(= -\frac{8}{5} + \frac{4}{7} - \frac{34}{35}\)
Samakan penyebut (KPK = 35):
\(= -\frac{56}{35} + \frac{20}{35} - \frac{34}{35}\)
\(= \frac{-56 + 20 - 34}{35} = \frac{-70}{35} = -2\) ✓
Jawaban: Berdasarkan perhitungan, \(x = -12\) dan \(y = -2\), sehingga \(x < y\). Namun tidak ada opsi yang tepat. Kemungkinan ada kesalahan pada opsi E atau perlu verifikasi ulang.

Soal 6
Diketahui nilai A sama dengan B
A B
\(4x - \frac{1}{5}y\) \(\frac{1}{3}x + y\)
Manakah hubungan yang benar antara kuantitas x dan y berikut berdasarkan informasi yang diberikan?
  • A. \(55x - 18y > 0\)
  • B. \(55x - 36y > 0\)
  • C. Hubungan x dan y tidak dapat ditentukan
  • D. \(55x - 18y = 0\)
  • E. \(55x - 36y = 0\)
Pembahasan:
Diketahui:
\(A = B\)
\(4x - \frac{1}{5}y = \frac{1}{3}x + y\)
Menyelesaikan persamaan:
\(4x - \frac{1}{5}y = \frac{1}{3}x + y\)
Pindahkan semua suku ke ruas kiri:
\(4x - \frac{1}{5}y - \frac{1}{3}x - y = 0\)
\(4x - \frac{1}{3}x - \frac{1}{5}y - y = 0\)
Samakan penyebut untuk x:
\(4x - \frac{1}{3}x = \frac{12x}{3} - \frac{x}{3} = \frac{11x}{3}\)
Samakan penyebut untuk y:
\(-\frac{1}{5}y - y = -\frac{1}{5}y - \frac{5}{5}y = -\frac{6}{5}y\)
Persamaan menjadi:
\(\frac{11x}{3} - \frac{6y}{5} = 0\)
Kalikan kedua ruas dengan KPK dari 3 dan 5, yaitu 15:
\(15 \times \frac{11x}{3} - 15 \times \frac{6y}{5} = 0\)
\(5 \times 11x - 3 \times 6y = 0\)
\(55x - 18y = 0\)
Memeriksa opsi jawaban:
D. \(55x - 18y = 0\) → Benar ✓
A. \(55x - 18y > 0\) → Salah (karena sama dengan 0)
B. \(55x - 36y > 0\) → Salah
C. Hubungan tidak dapat ditentukan → Salah
E. \(55x - 36y = 0\) → Salah
Jawaban: D. \(55x - 18y = 0\)

A B
\(\frac{3}{4}x - 10y\) \(\frac{6}{3}x - 5y\)
Manakah hubungan yang benar antara kuantitas A dan B berikut berdasarkan informasi yang diberikan? (TIU CPNS 2023)
  • A. \(3A = B\)
  • B. \(A = 2B\)
  • C. Hubungan A dan B tidak dapat ditentukan
  • D. \(5A > 3B\)
  • E. \(5A < 3B\)

Diketahui:
\(x = 4\)
\(y = \frac{4}{5}\)

\(A = \frac{3}{4}x - 10y\)
\(A = \frac{3}{4}(4) - 10\left(\frac{4}{5}\right)\)

Menghitung nilai B:
\(B = \frac{6}{3}x - 5y\)
\(B = 2x - 5y\)
\(B = 2(4) - 5\left(\frac{4}{5}\right)\)
\(B = 8 - 4 = 4\)
Membandingkan A dan B:
\(A = -5\)
\(B = 4\)
Jadi \(A < B\)
Memeriksa opsi jawaban:
A. \(3A = B\) → \(3(-5) = -15\), \(-15 = 4\)? Salah
B. \(A = 2B\) → \(-5 = 2(4) = 8\)? Salah
C. Hubungan tidak dapat ditentukan → Salah
D. \(5A > 3B\) → \(5(-5) = -25\), \(3(4) = 12\), \(-25 > 12\)? Salah
E. \(5A < 3B\) → \(5(-5) = -25\), \(3(4) = 12\), \(-25 < 12\)? Benar ✓
Jawaban: E. \(5A < 3B\)

Soal 10
Diketahui
M N
\(12\frac{1}{2} + 42 \times 1\frac{1}{2} - 3\) \(20\%\) dari 150
Manakah hubungan yang benar antara kuantitas M dan N berikut berdasarkan informasi yang diberikan?
  • A. \(2M < 3N\)
  • B. \(-3M > -2N\)
  • C. Hubungan M dan N tidak dapat ditentukan
  • D. \(\frac{1}{2}M < 2N\)
  • E. \(\frac{1}{3}M < \frac{1}{2}N\)
Pembahasan:

\(M = 12\frac{1}{2} + 42 \times 1\frac{1}{2} - 3\)

\(12\frac{1}{2} = \frac{25}{2} = 12,5\)
\(1\frac{1}{2} = \frac{3}{2} = 1,5\)
\(42 \times 1\frac{1}{2} = 42 \times \frac{3}{2} = 63\)
\(M = 12,5 + 63 - 3 = 72,5\)
Menghitung nilai N:
\(N = 20\%\) dari 150
\(N = \frac{20}{100} \times 150 = 0,2 \times 150 = 30\)
Membandingkan M dan N:
\(M = 72,5\)
\(N = 30\)
Jadi \(M > N\)
Memeriksa opsi jawaban:
A. \(2M < 3N\) → \(2(72,5) = 145\), \(3(30) = 90\), \(145 < 90\)? Salah
B. \(-3M > -2N\) → \(-3(72,5) = -217,5\), \(-2(30) = -60\), \(-217,5 > -60\)? Salah
C. Hubungan tidak dapat ditentukan → Salah
D. \(\frac{1}{2}M < 2N\) → \(\frac{1}{2}(72,5) = 36,25\), \(2(30) = 60\), \(36,25 < 60\)? Benar ✓
E. \(\frac{1}{3}M < \frac{1}{2}N\) → \(\frac{1}{3}(72,5) = 24,17\), \(\frac{1}{2}(30) = 15\), \(24,17 < 15\)? Salah
Jawaban: D. \(\frac{1}{2}M < 2N\)

  • D. \(X + Y = 0\)

Pembahasan:
Diketahui:
15 orang pekerja selesai dalam 10 hari

Konsep perbandingan berbalik nilai:
Semakin banyak pekerja, semakin cepat pekerjaan selesai (waktu semakin sedikit).
Jumlah pekerja × Waktu = Konstan
Menghitung nilai Y:
\(15 \times 10 = Y \times 5\)
\(150 = 5Y\)
\(Y = \frac{150}{5} = 30\) pekerja
Rumus perbandingan berbalik nilai:
\(\frac{P_1}{P_2} = \frac{W_2}{W_1}\)
Dimana \(P\) = jumlah pekerja, \(W\) = waktu
\(\frac{15}{Y} = \frac{5}{10}\)
\(15 \times 10 = Y \times 5\)
\(Y = 30\)
Nilai X:
\(X = 20\)
Membandingkan X dan Y:
\(X = 20\)
\(Y = 30\)
Jadi \(X < Y\)
Memeriksa opsi jawaban:
A. \(X > Y\) → \(20 > 30\)? Salah
B. \(X < Y\) → \(20 < 30\)? Benar ✓

\(57 = 3 \times 19\)

\(b = \frac{136}{399}\)
